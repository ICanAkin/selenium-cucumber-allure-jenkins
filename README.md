# Selenium Cucumber Automation Framework

## Tech Stack
- Java
- Selenium WebDriver
- Cucumber (BDD)
- TestNG
- Maven
- Allure Report
- Jenkins CI

## Test Scenarios
- Login functionality
- Product listing verification
- Add to cart validation

## Reporting
- Allure Report with failure screenshots
- Environment metadata support

## CI Integration
- Jenkins pipeline using Jenkinsfile
- Automated test execution on commit

## Run Tests
```bash
mvn clean test
